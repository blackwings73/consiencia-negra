# Consciência Negra — Site (Versão Visual Melhorada)

Este repositório contém um site estático de 3 páginas sobre Consciência Negra, com atenção a design, acessibilidade e performance.

## Como usar
- Abra `index.html` localmente ou hospede no GitHub Pages.
- Estrutura de pastas já organizada.

## Créditos
- Imagens SVG pequenas e embutidas como ilustrações de exemplo.
- Font: Inter (Google Fonts).

Substitua conteúdos de exemplo por imagens e referências reais antes de usar em trabalhos.
